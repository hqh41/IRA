.js Augmented Reality Component System

Class: ARViewer

ARViewer

Simple compositing viewer for augmented reality

Constructor

new ARViewer()

Slots

addScene(scene)

Adds new objects to the current 3D scene
Parameters:
Name Type Description
scene object 3D object as defined by Three.js to add to the scene.

keepAspectRatio(b)

Tells to keep the aspect ratio of the camera. It avoids deformations of the augmented scene.
Parameters:
Name Type Description
b boolean set it to true to keep aspect ratio, false otherwise

removeScene(scene)

Removes an object from the current 3D scene
Parameters:
Name Type Description
scene object 3D object as defined by Three.js to remove from the scene.

render()

Triggers the rendering of the composite scene

resetCamera()

Resets the position of the camera to the origin of the scene coordinate system

setExtrinsics(markers)

Set extrinsic camera parameters for the virtual camera
Parameters:
Name Type Description
markers array an array of markers

setFocal(focal)

Set the focal length of the virtual camera for very simple camera models.
Parameters:
Name Type Description
focal numeric focal length (in pixels) of the camera.

setIntrinsics(intrinsics)

Set intrinsic camera parameters for the virtual camera
Parameters:
Name Type Description
intrinsics array linearized array of camera parameters

setSceneId(id)

Set the scene id in case we should set extrinsic parameters.
Parameters:
Name Type Description
id number the id of the scene

setSize(width, height)

Sets the size of the viewport in pixels to render the scene
Parameters:
Name Type Description
width number width of the viewport
height number height of the viewport

setWidgets(cName, vName)

Initializes the widgets (HTML elements) needed as a basis for the viewer.
Parameters:
Name Type Description
cName string id of the container that will enclose the scene renderer
vName string id of the element at the source of the video stream

viewAll()

Computes the ideal camera position to see the whole 3D scene. Mainly useful for debugging
