Pose

Gives the pose of an object with respect to the camera. This object provides two estimates.
Type:
  • Object
Properties:
Name Type Description
bestError number reprojection error for best estimated pose
bestRotation Array.<number> rotation matrix (3x3 Array) of the best estimated pose
bestTranslation Array.<number> 3d coordinates of the translation vector
alternativeError number reprojection error for alternative estimated pose
alternativeRotation Array.<number> rotation matrix (3x3 Array) of the alternative estimated pose
alternativeTranslation Array.<number> 3d coordinates of the translation vector
