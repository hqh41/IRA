Class: Animator

Animator

A component that request new frames for animation. This component is useful when you want to create animations in the context of a web browser.

Constructor

new Animator()

Slots

start()

Starts requesting frames for animation. As soon as it is started, the signal onAnimationFrame is periodically triggered.
Emits signals:
  • onAnimationFrame

stop()

Stops requesting frames for animation.

Signals

onAnimationFrame()

Signals that an animation frame is ready.
