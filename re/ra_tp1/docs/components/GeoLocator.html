Class: GeoLocator

GeoLocator

Uses the location provider of the browser to determine the geolocation of the device. This component relies on the Geolocation API.

Constructor

new GeoLocator()

Slots

clearWatch()

Ends geolocation queries.
See:
  • watchPosition

getPosition()

Retrieves the geolocation of the device. Use this function for a one shot retrieval. If it succeeds, signals sendPosition and sendCoordinates are triggered. Otherwise, signal geolocationDisabled is triggered.
Emits signals:
  • sendPosition
  • sendCoordinates
  • geolocationDisabled

setTimeout(t)

Set the timeout (in milliseconds) between the call of getPosition or watchPosition and the sending of signal sendPosition.
Parameters:
Name Type Description
t number timeout duration in milliseconds

watchPosition()

Retrieves the geolocation of the device. Use this function if you want to continuously obtain the geolocation of the device. If it succeeds, signals sendPosition and sendCoordinates are triggered. Otherwise, signal geoLocationDisabled is triggered. You can use clearWatch to stop watching the position.
Emits signals:
  • sendPosition
  • sendCoordinates
  • geolocationDisabled

Signals

geolocationDisabled()

Notifies that the geolocation service is disabled

sendCoordinates(coords)

Sends the coordinates of the device using the coordinates object proposed in the specification of the geolocation API.
Parameters:
Name Type Description
coords object coordinates of the device.

sendPosition(lon, lat)

Sends the position of the device (longitude and latitude)
Parameters:
Name Type Description
lon number longitude of the geolocation
lat number latitude of the geolocation
