<static> surfDetDes(image, octaves, intervals, initSample, threshold)

Detects SURF features and extract their corresponding descriptors in a given image. Detection should be performed on gray images only. If it is not the case then use grayImage()
Parameters:
Name Type Description
image ImageData image into which to start the detection.
octaves number number of octaves on which to perform detection (default:5)
intervals number number of intervals on which to perform detection (default:4)
initSample number number of samples when initializing the algorithm (default: 2)
threshold number threshold to consider feature as a good feature (default: 0.0004)
Returns:
a list of points with their descriptors
