ARUCODetector

Component that detects ARUCO markers in images This component encapsulate the js-aruco library.

Constructor

new ARUCODetector()

Slots

detect(image)

Detects ARUCO markers in the given image. If markers are detected, this slot triggers the signal onMarkers.
Parameters:
Name Type Description
image obj the image in which markers should be detected
Emits signals:
  • onMarkers

Signals

onMarkers(markers)

Signal that is emitted when markers are detected in an image.
Parameters:
Name Type Description
markers Array.<Marker> Array of detected markers.
